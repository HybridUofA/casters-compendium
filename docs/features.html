Current features

A complete deckbuilding workflow.

Move from a searchable local card library to a saved, shareable, tournament-ready, or Tabletop Simulator-ready deck.

01

Flexible deck editing

Build main decks and sideboards with copy limits enforced. Select individual copies with Control-click or Command-click, move ordered batches within or between zones, and sort whenever you want the standard order back.

02

Full card search

Filter by name, type, trait, ability keyword, element, expansion, cost or level, and playtesting status. Preview artwork and rules text on hover.

03

Sharing and tournaments

Save editable JSON decks and exchange the human-readable format published by Speedrobo Games, including deck metadata, set names, totals, and sideboards.

04

Tabletop Simulator

Generate standalone image sheets or install a portable saved object in one click. Shared publisher-authorized card sheets load automatically for multiplayer participants, with local sheets retained as an offline fallback.

05

Local card library

Download verified card data and publisher-authorized artwork from the project's hosted catalog. Cached assets are shared by the deckbuilder and the future simulator.

06

Update awareness

A full-database SHA-256 detects approved catalog changes, while a manual update action keeps control in the player's hands.

07

Appearance settings

Follow the operating-system theme or select a persistent Light or Dark appearance, then choose the unobstructed interface or one of two bundled full-window backgrounds.

08

Cross-platform packages

Use precompiled packages for Windows, Intel and Apple Silicon Macs, Debian or Ubuntu, Arch Linux, and other x64 Linux systems.
