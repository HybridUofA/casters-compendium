Windows 10/11 · x64

Windows executable

Download .exe
  1. Move the downloaded file to a permanent folder if desired.
  2. Double-click it to launch—no installer is required.
  3. If SmartScreen appears, choose More info, verify the filename, then choose Run anyway.

A current OpenGL driver is required. Virtual machines may need guest graphics drivers or Mesa software OpenGL.

macOS · Apple Silicon

M1, M2, M3, M4, or later

Download ZIP
  1. Unzip the download.
  2. Move Caster's Compendium.app into Applications.
  3. For the first launch, Control-click the app, choose Open, then confirm.

macOS · Intel

Intel-based Macs

Download ZIP
  1. Unzip the download.
  2. Move Caster's Compendium.app into Applications.
  3. For the first launch, Control-click the app, choose Open, then confirm.

Debian / Ubuntu · x64

Native .deb package

Download .deb

Open a terminal in the download folder and run:

sudo apt install ./casters-compendium_0.1.3_amd64.deb

Arch Linux · x64

Native Arch package

Download package

Open a terminal in the download folder and run:

sudo pacman -U ./casters-compendium-0.1.3-1-x86_64.pkg.tar.zst

Other Linux · x64

Portable Linux archive

Download .tar.xz

Extract it, enter the directory, and install for your user:

tar -xJf CastersCompendium-linux-amd64.tar.xz
cd CastersCompendium
make user-install

For a system-wide installation, use sudo make install instead.
