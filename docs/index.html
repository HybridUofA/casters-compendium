Current features

Everything needed to move from an idea to a playable deck.

01

Flexible deck editing

Build main decks and sideboards with copy limits enforced. Drag cards to add, move, reorder, or remove them, then sort whenever you want the standard order back.

02

Full card search

Filter by name, type, trait, ability keyword, element, expansion, cost or level, and playtesting status. Preview artwork and rules text on hover.

03

Sharing and tournaments

Save editable JSON decks, import human-readable decklists, and export card-code text files suitable for sharing or tournament submission.

04

Tabletop Simulator

Generate image sheets for both the main deck and sideboard using the bundled card back, ready to import into Tabletop Simulator.

05

Local card library

Download card data, full artwork, and thumbnails concurrently. Cached assets are shared by the deckbuilder and the future simulator.

06

Update awareness

A deterministic card-list hash detects published database changes, while a manual update action keeps control in the player's hands.

Get Caster's Compendium v0.1.3

Download the package for your system directly—no GitHub release-page detour required. Windows and macOS packages are currently unsigned and may show a first-launch security prompt.

Windows 10/11 · x64

Windows executable

Download .exe
  1. Move the downloaded file to a permanent folder if desired.
  2. Double-click it to launch—no installer is required.
  3. If SmartScreen appears, choose More info, verify the filename, then choose Run anyway.

A current OpenGL driver is required. Virtual machines may need guest graphics drivers or Mesa software OpenGL.

macOS · Apple Silicon

M1, M2, M3, M4, or later

Download ZIP
  1. Unzip the download.
  2. Move Caster's Compendium.app into Applications.
  3. For the first launch, Control-click the app, choose Open, then confirm.

macOS · Intel

Intel-based Macs

Download ZIP
  1. Unzip the download.
  2. Move Caster's Compendium.app into Applications.
  3. For the first launch, Control-click the app, choose Open, then confirm.

Debian / Ubuntu · x64

Native .deb package

Download .deb

Open a terminal in the download folder and run:

sudo apt install ./casters-compendium_0.1.3_amd64.deb

Arch Linux · x64

Native Arch package

Download package

Open a terminal in the download folder and run:

sudo pacman -U ./casters-compendium-0.1.3-1-x86_64.pkg.tar.zst

Other Linux · x64

Portable Linux archive

Download .tar.xz

Extract it, enter the directory, and install for your user:

tar -xJf CastersCompendium-linux-amd64.tar.xz
cd CastersCompendium
make user-install

For a system-wide installation, use sudo make install instead.
